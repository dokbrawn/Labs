# README для проекта обработки бинарных файлов сетевых адаптеров
## Вариант 22

### Описание проекта

Данный проект реализует задание по учебной практике для студентов направления 
09.03.01 «Информатика и вычислительная техника», профиль «Программное обеспечение 
мобильных систем».

Проект состоит из двух программ:
1. **create_adapter** - программа создания бинарного файла adapter.dat с 20 записями
2. **process_adapter** - программа обработки бинарного файла с функциями поиска, 
   редактирования и отображения данных

### Структура данных

Каждая запись в файле adapter.dat имеет следующую структуру:
- Тип сетевого адаптера (50 символов)
- Интерфейс подключения (30 символов)
- Скорость передачи данных (int, Мбит/с)
- Количество портов (int)
- Чип (50 символов)
- Стандарт USB (20 символов)

Общий размер одной записи: 160 байт
Количество записей: 20
Общий размер файла: 3200 байт

### Требования к компиляции

- Компилятор: g++ (GCC) версии 4.8 или выше
- Стандарт C++: C++11 или выше
- Операционная система: Linux, Windows (через MinGW), macOS

### Сборка проекта

#### Использование Makefile (Linux/macOS):
```bash
make          # Сборка всех программ
make clean    # Очистка скомпилированных файлов
make rebuild  # Пересборка
```

#### Ручная компиляция:
```bash
# Компиляция программы создания
g++ -std=c++11 -Wall -Wextra -o create_adapter create_adapter.cpp

# Компиляция программы обработки
g++ -std=c++11 -Wall -Wextra -o process_adapter process_adapter.cpp
```

#### В Windows (MinGW):
```bash
g++ -std=c++11 -Wall -Wextra -o create_adapter.exe create_adapter.cpp
g++ -std=c++11 -Wall -Wextra -o process_adapter.exe process_adapter.cpp
```

### Запуск программ

#### 1. Создание бинарного файла:
```bash
./create_adapter      # Linux/macOS
create_adapter.exe    # Windows
```

Программа создаст файл adapter.dat с 20 записями о сетевых адаптерах.

#### 2. Обработка бинарного файла:
```bash
./process_adapter     # Linux/macOS
process_adapter.exe   # Windows
```

### Функционал программы обработки

Программа process_adapter предоставляет следующее меню:

1. **Показать все записи** - отображает все 20 записей из файла
2. **Поиск по типу адаптера** - поиск записей по введенному типу (регистронезависимый)
3. **Редактировать запись** - изменение данных выбранной записи
4. **Сохранить изменения** - сохранение отредактированных данных обратно в файл
5. **Информация о файле** - отображение характеристик файла (размер, количество записей)
6. **Информация о разработчике** - данные о студенте и варианте задания
0. **Выход** - завершение работы программы

### Примеры использования поиска

- Поиск "Wi-Fi" найдет 8 записей с Wi-Fi адаптерами
- Поиск "Bluetooth" найдет 4 записи с Bluetooth адаптерами
- Поиск "Ethernet" найдет 6 записей с Ethernet адаптерами
- Поиск несуществующего типа выдаст сообщение об отсутствии результатов

### Примечания

1. Перед запуском process_adapter необходимо сначала запустить create_adapter 
   для создания файла adapter.dat
2. При редактировании записей можно оставить поле пустым, чтобы сохранить 
   предыдущее значение
3. Поиск выполняется регистронезависимо
4. Все строковые поля имеют фиксированный размер с нулевым завершением

### Возможные ошибки и способы их устранения

1. **"Ошибка: не удалось открыть файл adapter.dat"**
   - Решение: Запустите сначала программу create_adapter для создания файла

2. **"Ошибка записи в файл"**
   - Решение: Проверьте права доступа к директории и наличие свободного места

3. **"Неверный номер записи"**
   - Решение: Введите число от 1 до 20
